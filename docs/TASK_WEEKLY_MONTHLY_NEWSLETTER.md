# 주간/월간 뉴스레터 확장 및 아카이빙 프로젝트

## 📋 프로젝트 개요

- **목표**: 데일리 뉴스레터를 기반으로 주간/월간 뉴스레터를 자동 생성 및 발송하고, 모든 발송 이력을 체계적으로 아카이빙합니다.
- **기간**: 2025-12-22 ~ (예상 소요 시간: 12-16시간)
- **주요 기능**:
  - 데일리 뉴스레터 HTML 파일 자동 저장
  - 주간 뉴스레터 자동 생성 (전주 데이터 기반, 매주 월요일)
  - 월간 뉴스레터 자동 생성 (전월 데이터 기반, 매월 첫 영업일)
